## 스마트 홈 IoT 시스템 설계

### Part A - 시스템 구성 요소
전원 제어, 보안 감시, 환경 모니터링이 전부 가능한 시스템
#### 센서
- DHT22(온도, 습도 센서):실내 온도와 습도를 측정 -> 환경 모니터링에 사용
- PIR(모션 센서): 움직임을 감지하여 사람의 침입을 확인 -> 보안 감시에 사용
- MQ-2(가스 센서): 가스 누출을 감지하여 
*이 부분에서 센서의 이름이 생각나지 않아 AI의 도움을 받음*
#### 액추에이터
- 릴레이 모듈 (+팬, 조명): ESP32의 명령을 받아서 전원을 켜고 꺼서 전원제어를 담당
#### ESP32 장치
- ESP32 제어장치:센서 데이터를 읽어서 MQTT로 발행하고, 대시보드의 제어 명령을 구독하여 액추에이터를 작동시키는 컨트롤러
#### MQTT 브로커
- ESP32와 대시보드 사이에서 메세지를 중계, 센서 데이터 전송과 제어 명령 전달 
#### 대시보드
- Node-RED 대시보드: 센서 데이터를 게이지와 차트로 시각화, 사용자가 버튼으로 장치를 제어할 수 있게 해주는 시각적 장치

### Part B - 시스템 다이어그램
![시스템 다이어그램](https://imgur.com/a/Qej6pqP.png)
**데이터 흐름**
- 센서 3개가 주변 환경 값을 측정
- ESP32가 센서 값을 읽고 MQTT 메세지로 변환하여 발행
- MQTT 브로커가 메세지를 받아서 중계
- Node-RED 대시보드가 구독하여 게이지와 차트 값을 표시

**제어 명령**
- 사용자가 대시보드 스위치를 누르면 명령 ON/OFF를 발행
- MQTT 브로커가 메세지를 중계
- ESP32가 명령을 받음 -> 릴레이에 작동 신호를 보내어 팬/조명을 키거나 끈다
